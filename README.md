# 📝 Go CLI To-Do App (Day 2)

A simple and persistent **CLI-based To-Do List Application built using Go (Golang)** to understand file handling and real-world CLI usage.

This project allows users to add, view, and clear tasks, with data stored in a local file.

This project is part of my **Go learning journey (Day 2 Project)**.

---

## 🚀 Features

* Add new tasks
* View all tasks
* Persistent storage using file system
* Clear all tasks
* Simple CLI interface

---

## 🛠 Built With

* **Go (Golang)**
* Standard Library (os, bufio, fmt)
